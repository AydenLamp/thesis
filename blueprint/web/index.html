Green’s preorders are traditionally defined for semigroups by adjoining a unit element and working in the resulting monoid. Here we start directly with monoids, thereby avoiding the overhead of the ‘WithOne‘ construction and the need to explicitly adjoin an identity. Since a semigroup with an adjoined unit is essentially the same as a monoid, working on monoids simplifies many of the subsequent proofs. In future developments this code might be refactored to operate on semigroups with a unit, but the monoid formulation suffices for our purposes.

Definition 1 Green’s R-preorder
#

Let \(M\) be a monoid and let \(x,y\in M\). We define \(x \le _R y\) if there exists \(z\in M\) such that \(x = y\cdot z\). Equivalently, \(x\) lies in the principal right ideal generated by \(y\).

Lemma 2 Reflexivity of \(\le _R\)
#

Uses

  • Definition 1

For every \(x\in M\), \(x \le _R x\).

Lemma 3 Transitivity of \(\le _R\)
#

Uses

  • Definition 1

For all \(x,y,z\in M\), if \(x \le _R y\) and \(y \le _R z\) then \(x \le _R z\).

Proof ▶

Suppose \(x \le _R y\) and \(y \le _R z\). By definition there exist \(v,u\in M\) with \(x = y\cdot v\) and \(y = z\cdot u\). Taking the witness \(u\cdot v\) we compute \(z\cdot (u\cdot v) = (z\cdot u)\cdot v = y\cdot v = x\), so \(x \le _R z\).
